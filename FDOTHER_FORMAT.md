# FDOTHER.DAT 格式分析文档

## 概述
FDOTHER.DAT 是一个混合资源文件，包含多种类型的图像数据（字体、UI、子画面等）。文件大小: 3,382,481 字节，共52个索引。

## 索引结构
- **头部**: 字节0-5 = `LLLLLL` (6字节)
- **索引表**: 从字节6开始，每索引8字节 = (start_uint32 + end_uint32)
- **资源定位**: `start` 和 `end` 是相对于文件开头的偏移

## 资源类型分类

**特征**: 前4字节是 width(uint16) + height(uint16)，后续是RLE编码数据

### 2. LLLLLL类型 (14个索引)
**特征**: 字节0-5 = `LLLLLL`，字节6-7 = offset_count，然后是offset表

**索引**: 6, 24, 25, 26, 32, 39, 40, 41, 42, 43, 44, 45, 46, 47

**结构**:
- 字节0-5: `LLLLLL` 头
- 字节6-7: offset_count (uint16)
- 字节8+: offset表 (每entry 2字节 uint16)
- offset表后: 实际数据区域

**注意**: 这类资源的offset表结构需要进一步分析

### 3. LMI1类型 (2个索引)
**特征**: 字节0-3 = `LMI1`，后续是offset表

**索引**: 3, 7

**结构**: 看起来是类似指针表的结构，指向其他数据

### 4. 尺寸异常资源 (8个索引)
**索引**: 0, 1, 2, 4, 38, 51

**可能原因**: 
- width/height值存储在其他位置
- 或者这些不是图像资源
