## RLE 解码格式
函数 `sub_4E98D` 实现了 RLE 解码：

### 数据头
```
struct RLEHeader {
    WORD width;      // 宽度
    WORD height;     // 高度
    BYTE data[];     // RLE 编码数据
};
```

### 编码规则
控制字节高位决定操作类型：

| 控制字节范围 | 操作 |
|-------------|------|
| 0x00-0x3F | 跳过 N+1 像素 (透明) |
| 0x40-0x7F | 复制 N+1 字节到目标 |
| 0x80-0xBF | 用下一个字节填充 N+1 像素 |
| 0xC0-0xFF | 交替填充 N+1 像素对 |
