# FDOTHER.DAT 格式文档 v3

## 文件概览
- 大小: 3,382,481 字节
- 资源数: 103个 (4字节索引, 偏移6开始)
- 格式: 通用DAT容器 (sub_111BA加载)

## 索引表格式
基于逆向 sub_111BA: `fseek(file, 4 * index + 6, 0)`
- 每项4字节 (uint32, 小端序)
- 相邻项组成 (start, end) 对
- start = 当前项值, end = 下一项值  
- 大小 = end - start

### 2. RLE压缩图像
基于逆向 sub_4E98D 的4模式RLE:
- 头部: uint16 width + uint16 height
- 2位命令码:
  - 00 (0x00-0x3F): 原始数据, count=(cmd&0x3F)+1 字节
  - 01 (0x40-0x7F): 单色填充, count=(cmd&0x3F)+1, 读1字节重复
  - 10 (0x80-0xBF): 跳过/透明, count=(cmd&0x3F)+1 像素
  - 11 (0xC0-0xFF): 隔像素写入, count=(cmd&0x3F)+1, 读1字节, 隔1像素写

### 3. LMI1动画
基于逆向 sub_16886: `*(DWORD*)(data + 4*frame + 6) + data`
- 头部: "LMI1" (4B) + frame_count (uint16) + first_frame_offset (uint16)
- 偏移6开始: 4字节/帧的偏移索引表
- 每帧 = 独立RLE图像 (同sub_4E98D格式)
